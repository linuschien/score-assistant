# US-08 加權總分計算與匯出 (Weighted Score Calculation & Export)

## 背景 (Background)

教師需要系統依照各成績項目的權重，自動計算每位學生的加權總分，並在匯出前以全班成績預覽表確認內容正確，最後再匯出至試算表格式以供存檔或進一步分析。

---

## US-08-00：預覽全班成績總表

**身份**：教師 (Teacher)

> **As a** 教師，  
> **I want to** 在匯出前於系統內預覽全班成績總表，  
> **So that** 我可以確認所有成績與加權總分無誤後，再決定是否匯出。

### 驗收條件 (Acceptance Criteria)

- **AC1**：系統提供「成績總表預覽」頁面，以二維表格呈現全班成績：
  - **縱軸（列）**：每位學生佔一列，依座號升冪排列，顯示座號與姓名。
  - **橫軸（欄）**：每個成績項目佔一欄，欄位標題顯示項目名稱（可加上日期，如「課堂表現 05/10」）。
  - **最後一欄**：為該生的加權總分。
- **AC2**：表格標題列與座號姓名欄固定凍結（Sticky Header & Sticky Column），支援左右與上下捲動，方便瀏覽大量資料。
- **AC3**：尚未登記成績的儲存格以特殊樣式標示（如灰色底色 + 「-」），與已登記的 0 分明確區分。
- **AC4**：`CLASSROOM_PERFORMANCE` 類型的負分儲存格以紅色文字標示。
- **AC5**：表格頂部顯示彙總資訊列：全班平均加權總分、最高分學生、最低分學生。
- **AC6**：若總權重不等於 100%，預覽頁面頂部顯示明顯警告橫幅，提示教師先修正權重設定。
- **AC7**：預覽頁面提供「匯出」按鈕，點擊後直接進入 US-08-03 的匯出流程，無需重複操作。

---

## US-08-01：計算全班加權總分

**身份**：教師 (Teacher)

> **As a** 教師，  
> **I want to** 讓系統自動計算每位學生的加權總分，  
> **So that** 我不需要手動計算，能快速得到最終學期成績。

### 驗收條件 (Acceptance Criteria)

- **AC1**：系統依以下公式計算加權總分：
  ```
  加權總分 = Σ（單項成績 / 單項滿分 × 單項權重）
  ```
- **AC2**：加權總分的計算結果以小數點後兩位呈現（四捨五入）。
- **AC3**：若某學生某項目尚未登記成績，該項目以 0 分計入加權（並在介面上標示「未登記」）。
- **AC4**：若所有成績項目的權重總和不等於 100%，系統在總分欄位旁顯示警告圖示，提示計算結果可能不準確。
- **AC5**：加權總分頁面顯示所有學生的座號、姓名、各項目得分、加權總分，並依加權總分排序。

## US-08-02：查看成績摘要統計

**身份**：教師 (Teacher)

> **As a** 教師，  
> **I want to** 查看全班加權總分的統計摘要，  
> **So that** 我可以快速了解班級整體學習成效。

### 驗收條件 (Acceptance Criteria)

- **AC1**：系統顯示以下統計數字：班級平均加權總分、最高分、最低分、通過率（以 60 分為預設門檻，可調整）。
- **AC2**：系統以長條圖呈現成績分布（如：0-59、60-69、70-79、80-89、90-100 各區間人數）。

## US-08-03：匯出成績報表 (Excel / CSV)

**身份**：教師 (Teacher)

> **As a** 教師，  
> **I want to** 將全班成績（含各項目分數與加權總分）匯出為 Excel 或 CSV 檔案，  
> **So that** 我可以進行存檔、繳交行政或進一步的資料分析。

### 驗收條件 (Acceptance Criteria)

- **AC1**：教師可以選擇匯出格式：Excel（`.xlsx`）或 CSV（`.csv`）。
- **AC2**：匯出的欄位結構與 US-08-00 預覽總表一致：座號、姓名、各成績項目分數（每項一欄）、加權總分。
- **AC3**：匯出的 Excel 檔案標題行與座號姓名欄鎖定凍結（Freeze Panes），方便捲動查看大量資料。
- **AC4**：檔案名稱預設為：`[學期名稱]_[班級名稱]_成績報表_[匯出日期].xlsx`（例：`113學年度第一學期_三年甲班_成績報表_20250510.xlsx`）。
- **AC5**：匯出過程中若發生錯誤，系統顯示明確的錯誤訊息。
- **AC6**：教師可直接從 US-08-00 預覽頁面觸發匯出，亦可從選單獨立進入匯出功能。

## US-08-04：匯出出缺席摘要報表

**身份**：教師 (Teacher)

> **As a** 教師，  
> **I want to** 單獨匯出出缺席紀錄的摘要報表，  
> **So that** 我可以配合學校行政要求提交出缺席統計。

### 驗收條件 (Acceptance Criteria)

- **AC1**：教師可以選擇只匯出類型為 `ATTENDANCE` 的成績項目。
- **AC2**：匯出欄位包含：座號、姓名、各出缺席日期（每日一欄）、出席次數、缺席次數、請假次數。
- **AC3**：支援 Excel（`.xlsx`）格式匯出。
